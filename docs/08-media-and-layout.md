# Медиа-запросы и вёрстка: кратко

## Mobile-first

Сначала стили для **узкого экрана**, затем расширение через `min-width`:

```css
.grid {
  display: grid;
  gap: 1rem;
  grid-template-columns: 1fr;
}
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

Типичные контрольные точки (ориентир): **768px** (планшет), **1024px** (десктоп).

## Гибкая типографика

Функция **`clamp()`** задаёт минимум, предпочтительное и максимальное значение размера шрифта без лишних медиа-запросов:

```css
h1 {
  font-size: clamp(1.25rem, 2vw + 1rem, 2rem);
}
```

## Каталог карточек

- **CSS Grid** — удобно для равных карточек в сетке.
- **Flexbox** — для строк с выравниванием по высоте внутри карточки.

Ограничьте ширину контента: `max-width` + горизонтальные отступы (`padding-inline`), чтобы на широких мониторах текст не растягивался чрезмерно.

## Итог

Единый контейнер, сетка карточек с перестроением колонок и читаемые размеры шрифтов — база для «аккуратного» интерфейса без перегруза медиа-запросами.
